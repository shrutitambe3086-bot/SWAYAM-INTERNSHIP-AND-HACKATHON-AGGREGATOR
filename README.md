# 🚀 Swayam – AI-Powered Student Growth Platform

Swayam is an intelligent platform designed to help students grow, learn, and track their progress efficiently. It combines **AI-driven recommendations, skill-based learning tracks, and real-time analytics** to guide users toward better career opportunities.


## 🌟 Features

* 🎯 **Personalized Learning Tracks**
  Get curated learning paths based on your interests and skills.

* 🤖 **AI Recommendations**
  Smart suggestions for courses, hackathons, and internships.

* 📊 **Progress Tracking Dashboard**
  Visual insights into your learning journey and achievements.

* 🏆 **Gamification System**
  Earn XP, badges, and track your growth.

* 💳 **Subscription & Payments**
  Upgrade plans with secure payment flow (UPI-based system).

* 🔐 **Authentication System**
  Secure login using Firebase Authentication (Google Sign-In supported).

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend / Database:** Firebase (Firestore)
* **Authentication:** Firebase Auth
* **Payments Logic:** Custom UPI-based simulation
* **Hosting:** Local / Firebase Hosting

## 💡 How It Works

1. User signs in using Google
2. Dashboard loads personalized content
3. User selects learning tracks
4. Progress is tracked and stored
5. User can upgrade subscription plans
6. Payment data is stored in Firestore
